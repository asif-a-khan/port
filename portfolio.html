Zoltar's Maze

React JS/ES6 Javascript | SASS/CSS3 | HTML5 | ( Live Site ) | ( Github Repo )

Brace yourself for a mystical journey through Zoltar's maze. This app pays omage to the movie 'Big' - 1998. Keeping true to Zoltar's ways, the user must make a wish and navigate through Zoltar's game in order to have their wish granted. This app uses the advice slip API to give users advice related to their wish upon completing the maze.

Created by: Asif Khan, Prachi Gujar, Barrie Wu and Tenal Bourchier

Pokedex

Jquery/ES6 Javascript | SASS/CSS3 | HTML5 | ( Live Site ) | ( Github Repo )

Yeah I know, why does the largest franchise in the world need another app? Well this app was designed to be a catalogue of all pokemon from generation 1-7. Users can search for any pokemon they would like and have its' stats displayed alongside an image of the pokemon. The user also has the option to checkout related pokemon.

Created by: Asif Khan and Calvin Barrett

Gilmore Girls API

React JS | Express JS | Node JS | MongoDB | ( Live site ) | ( Front End Repo ) ( Backend End Repo )

This app was designed to host pop culture references made in the show 'Gilmore Girls'. For viewers of the show it's not a surprise if some of these references fly over their head because, they happen so quick. Users of this app can browse through each episode and view/add references made in those episodes. An admin must login and approve any references inputted by users.

Created by: Asif Khan, Greg Forster, Caitlin Klotz and Boris Ulanowicz
